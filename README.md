# Try React hooks

## :hammer: Install

```bash
# install dependencies
yarn

# serve with hot reload at localhost:3000
yarn dev

# build for production with minification
yarn deploy
```

## :boat: Demo(if not open use vpn)

https://awesome-react-hooks.netlify.com/

- [React](https://reactjs.org/)
